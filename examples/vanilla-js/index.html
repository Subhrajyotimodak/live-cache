LiveCache Core APIs

Vanilla JavaScript example using core primitives via the UMD build (LiveCache global). Logic is split into separate JS files loaded in order.

1) Create / Insert

Uses a UserController registered in the default ObjectStore, and writes through the controller (publish/subscribe friendly).

2) Query / Update / Delete

Controller-driven updates publish changes. Deletes are performed on the underlying Collection.

3) Serialize / Hydrate

Hydration replaces the in-memory state of the collection.
